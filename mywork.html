Responsive Image Gallery

This project is a Responsive Image Gallery. It uses an XHR request to load data from Unsplash's API. This is a choosen query search with the topic of kittens. This gallery uses animations whenever the cat icon is clicked and whenever they photo is hovered on. I really enjoyed doing this website because I liked working with animations to get the exact look and feel that was wanted by the instructor. While I have had some issues really understanding how media queries work before this site, I finally was able to grasp the concept with this site. The site is responsive so it works on mobile, tablets and desktops. If you would like to view this site you can go to this link.

Responsive Form

This project is a Responsive Order Gallery. This project is to showcase my use of JavaScript to validate the user input forms. I also used JavaScript to auto update the order summary section as the user types based on the keypress event. This site is responsive working on a range of devices. I have also learned on working with and styling radio buttons to make the tshirt color and size options. Since I was struggling with the responsive side of things when creating this project I still have to fix an issue with its size on a desktop. Movie Search

This project is a site that allows the user to search for a movie returning the movie poster. The user is allowed to search for a movie using the input box. Their response is then stored into a variable to add to the search query. The API I am calling the data from is The Movie Database. This was one of my first sites dealing with CSS.
